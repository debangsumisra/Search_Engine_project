

##  How to Use

## Scrape One Webpage

**Format:**

```bash
python try.py <url>
```

**Example:**

```bash
python try.py https://www.wikipedia.org
```

---

###  Compare Two Webpages

**Format:**

```bash
python try.py <url1> <url2>
```

**Example:**

```bash
python try.py https://www.wikipedia.org https://example.com
```

---
